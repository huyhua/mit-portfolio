# Hứa Vương Duy Bình Portfolio

Vietnamese-first portfolio website built with Next.js, TypeScript, and Tailwind CSS for Vercel deployment.

## Development

```bash
pnpm install
pnpm dev
```

## Validation

```bash
pnpm lint
pnpm build
```

## Deploying to Vercel

Import this folder into Vercel as a Next.js project, or deploy from an authenticated CLI session:

```bash
vercel --prod
```
